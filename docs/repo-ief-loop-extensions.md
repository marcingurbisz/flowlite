# FlowLite repo IEF loop extensions

These rules extend the base IEF loop described in the workspace-level `AGENTS.md`.

## [WHEN REQUESTED] Exploratory tests

Do exploratory tests using our local `runPerfTestApp`.
Search for bugs and performance issues.
Shortly document what you have tested as an item in `TODO.md`.
If you find bugs, add them as TODO items in `TODO.md` and fix them immediately when they are in scope for the current loop.

## [EVERY LOOP] Review your own changes

Review your own changes from the current loop, looking for potential improvements or simplifications in both the code and the surrounding concepts.
Report any finding as a new TODO item with the `[FOR HUMAN REVIEW]` marker.
